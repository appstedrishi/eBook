Lateral line system

lateral line system

A system found along both sides of most fishes and aquatic amphibians that contains mechanoreceptors that detect low-frequency waves by a mechanism similar to the function of the inner ear.

Definition of a lateral line system

A system found along both sides of most fishes and aquatic amphibians that contains mechanoreceptors that detect low-frequency waves by a mechanism similar to the function of the inner ear.
