trophic Cascade model

A top-down model that postulates that it is mainly predation that controls community organization because predators control herbivores, which in turn control plants, which in turn control nutrient levels.

Definition of trophic Cascade model

A top-down model that postulates that it is mainly predation that controls community organization because predators control herbivores, which in turn control plants, which in turn control nutrient levels.

trophic Cascade model is a type of: ecological model.
